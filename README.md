# NetAnalyst Front

Frontend do [NetAnalyst](https://github.com/JFellipeLima/NetAnalyst) — sistema de monitoramento de domínios.

## Sobre

Interface gerada com [Lovable](https://lovable.dev) para fins de aprendizado. A integração com a API e o tratamento dos dados foram desenvolvidos manualmente.

## Tecnologias

## Como rodar

```bash
git clone https://github.com/JFellipeLima/NetAnalyst-Front.git
cd NetAnalyst-Front
npm install
npm run dev
```
